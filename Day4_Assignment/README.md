# Day 4 Assignment Pack

This folder contains a complete assignment set for Day 2 + Day 3 topics.

## What this assignment covers
- Day 2 concepts: pandas, NumPy-style data analysis, filtering, grouping, summarization.
- Day 3 concepts: classes, objects, inheritance, polymorphism, type hints, dataclass, factory, singleton, and a simple pipeline design.

## Folder structure
- data/healthcare_assignment_dataset.csv
  - This is the dataset the candidate will use.
- basic/assignment_basic.py
  - Simple tasks for beginners.
- intermediate/assignment_intermediate.py
  - Practical data analysis and OOP practice.
- advanced/assignment_advanced.py
  - Higher-level analytics and object-oriented pipeline design.

## Candidate instructions
1. Read the dataset first.
2. Complete the tasks in the basic file.
3. Move to the intermediate file.
4. Finish the advanced file for final evaluation.

## Expected learning outcome
By the end of this assignment, the candidate should be able to:
- load and inspect a real dataset,
- compute simple business metrics,
- use functions and classes to organize code,
- explain the difference between inheritance and polymorphism,
- and build a simple analytics pipeline.

## Suggested evaluation points
- Correct dataset loading and cleaning
- Correct cost and visit calculations
- Clear use of functions and classes
- Proper comments and readable code
- Correct output files
