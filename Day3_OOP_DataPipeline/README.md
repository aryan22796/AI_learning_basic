# Day 3 OOP DataPipeline

This folder contains the Day 3 notebook for OOP, type hints, dataclasses, factory/singleton patterns, and a typed DataPipeline example.

## Files
- `Day3_OOP_Basics.ipynb` — beginner-friendly notebook with simple explanations of classes, `self`, `__init__`, inheritance, polymorphism, type hints, and dataclasses.
- `Day3_OOP_DataPipeline.ipynb` — main training notebook with more examples and the typed pipeline lab.
- `data/healthcare.csv` — healthcare dataset used in the examples
